# GitHub Copilot Dev Day | Gombe

This is the official website for GitHub Copilot Dev Day | Gombe.

## Hosting on GitHub Pages

1. Create a new repository on GitHub (e.g., `github-copilot-dev-days-gombe`).
2. Push this code to the repository.
3. Go to the repository settings.
4. Under "Pages", set the source to "Deploy from a branch" and select the main branch.
5. The site will be available at `https://yourusername.github.io/github-copilot-dev-days-gombe/`.

## Updating Links

- Replace placeholder links in `index.html` with actual URLs:
  - Sweepstakes form
  - Pictures folder or gallery URL
  - Google Sheet URL
  - Download link for the data app

## Adding Assets

- Place event pictures in the `images/` folder.
- Place the data app zip in the `downloads/` folder.
