# GitHub Copilot Dev Days | Gombe

This is the official website for GitHub Copilot Dev Days | Gombe.

## Hosting on GitHub Pages

1. Create a new repository on GitHub (e.g., `github-copilot-dev-days-gombe`).
2. Push this code to the repository.
3. Go to the repository settings.
4. Under "Pages", set the source to "Deploy from a branch" and select the main branch.
5. The site will be available at `https://yourusername.github.io/github-copilot-dev-days-gombe/`.

## Setup for Embedded Sheet

To enable the embedded Google Sheet on the site:
1. Open the Google Sheet: https://docs.google.com/spreadsheets/d/1DZm1uKbJam_ZzAft5B2Mdn68Ke_mSnIqhlLNJIVhOig/edit?usp=sharing
2. Click "File" > "Share" > "Publish to web".
3. Select "Entire Document" and "Web page", then "Publish".
4. The embed will work on the site.
